# Navi Kalam (ਨਵੀਂ ਕਲਮ / नई कलम) 
A responsive, mobile-first Progressive Web App (PWA) designed for localized digital learning, featuring hybrid multi-language workflows, a real-time drawing whiteboard, and automated time-benchmarked evaluation tracking.
Navi Kalam is an interactive, mobile-first digital learning companion built to facilitate accessible education. Designed primarily for localized environments, it bridges learning gaps by shifting seamlessly between multiple languages (English, Punjabi, and Hindi) and providing tools that support both online classrooms and completely offline peer-to-peer data distribution.

## ✨ Core Features

*   **Localized Translation Matrix:** Complete contextual app localization supporting English, Punjabi, and Hindi dynamically at runtime.
*   **Frictionless Workspace Entry:** Instantly scales custom interactive dashboards tailored uniquely for Student study metrics or Teacher curriculum management without intrusive entry barriers.
*   **Digital Whiteboard Playground:** A high-performance canvas engine with adjustable markers, full color swatches, structural path undo arrays, and raster graphic export downloads for tracking diagram assignments.
*   **Time-Benchmarked Evaluation Tracking:** Randomized 10-question subject practice exams featuring built-in countdown timers and an advanced progress report matrix tracking precise down-to-the-second pacing for every individual question.
*   **Localized WebRTC Sync Hub:** An offline peer-to-peer networking bridge that leverages standard high-density QR camera scanning to establish independent data transfers between teachers and students without active internet routing.
*   **Progressive Web App (PWA) Baseline:** Full service worker caching architecture supporting offline launch-readiness, translucent mobile framing, and stand-alone home screen app shortcuts.

## 🛠️ Built With

*   **Frontend UI Shell:** Pure HTML5, utility-first Tailwind CSS classes, and FontAwesome visual iconography assets.
*   **Application Controller Tier:** Vanilla JavaScript (ES6 Modules) driving reactive state management architectures.
*   **Offline Data Engines:** Standard Browser LocalStorage APIs and custom Service Worker cache intercepts.
*   **Computer Vision Frameworks:** Integrates `Html5Qrcode` scanner libraries paired against native platform BarcodeDetector pipelines.
